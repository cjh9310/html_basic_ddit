추가 메소드
	append(), prepend() - 대상 요소의 안쪽 앞뒤로 추가
	before(), after() - 대상 요소의 바깥쪽 앞뒤로 추가
	
	*appendChild() 와 append(), appendTo() 차이점
	- appendChi;d() : 노드객체만 사용가능              
					: 한번에 하나의 노드만 추가 가능   
					: return값을 반환
	
	- append,To()   : 노드객체나 DomString(text) 사용 가능
					: 한번에 여러 개의 자식 요소 추가 가능
					: return값을 반환하지 않음
